What is Project Chrono?

ProjectChrono is a set of open-source tools for mechanical simulations. The core of ProjectChrono is the Chrono::Engine C++ middleware, a powerful object-oriented set of libraries whose application program interface (API) can be freely used to develop simulation software. The Chrono::Engine project is based on a platform-independent, open-source design.
